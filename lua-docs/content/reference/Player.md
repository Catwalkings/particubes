# Player

The Player type describe a human player, playing the game as well as its ingame character/avatar.

- Caps fields are read-only (expect for exceptions)
- Lowercase fields are free to use (read/write)

## Fields

### Id (Integer)

### IsLocal (Boolean)

### IsOnGround (Boolean)

### Jump (Number3)

### Mode (PlayerMode)

### Position (Number3)

### Rotation (Number3)

### Say (Function)

### Velocity (Number3)
